# OpenAI Status Bot
A Telegram bot to track the OpenAI service status and send notifications when theres a status update.

## How to
1. Clone this repo
2. Make a `.env` file and write `TOKEN=<your telegram bot token>`
3. Start the bot with `npm run start`
